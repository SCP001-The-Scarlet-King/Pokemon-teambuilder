# Competitive Pokémon Team Builder

## Overview

This Django web application allows users to create and save teams for Competitive Pokémon battles.

## Installation

Follow these steps to set up the project locally:

1. Clone the repo.
2. Install all the dependencies, by typing the following inside ur terminal:
   `pip install -r requirements.txt`
3. Run the following:
   `python manage.py migrate`
4. Run the server:
   `python manage.py runserver`
   Go to http://localhost:8000 to view the application.
